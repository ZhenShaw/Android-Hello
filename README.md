# Android-Hello
第一个安卓小程序。

### 涉及知识点
- [x] LinearLayout基础页面布局
- [x] 页面转跳及传参
- [x] 简单的动画实现
- [x] 用户注册登录功能
- [x] 使用OkHttp进行POST请求
- [x] json简单解析
- [x] Android6.0+权限申请
- [x] 蓝牙设备扫描
- [ ] 蓝牙连接
- [ ] 蓝牙通讯及文件传输


* 其中注册登录功能由后端提供，主要使用 GoLang + PostgreSQL。
